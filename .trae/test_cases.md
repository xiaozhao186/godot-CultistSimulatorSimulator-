# 测试用例文档（底部UI栏与设置面板）

## 1. 暂停按钮

- 点击暂停：游戏进入暂停状态，按钮高亮
- 再次点击：恢复运行状态，按钮取消高亮
- 设置面板打开时：暂停按钮禁用，避免状态冲突

## 2. 调速按钮

- 初始为 1x：显示默认图标
- 点击切换到 2x：显示加速标识图标，Engine.time_scale 变为 2
- 再次点击回到 1x：Engine.time_scale 变为 1
- 设置面板打开时：调速按钮禁用

## 3. 设置面板开关与暂停联动

- 点击设置按钮：面板打开；自动暂停；遮罩出现；面板居中显示
- 点击关闭（×）或再次点击设置按钮：面板关闭；恢复打开前的暂停状态

## 4. 分辨率切换

- 下拉选择 800×600 / 1280×720 / 1920×1080 / 2560×1440
- 选择后窗口尺寸立即切换到对应分辨率
- 切换失败时应记录错误（通过 settings_save_failed/settings_load_failed 信号上报）

## 5. 持久化

- 改变分辨率后重启：默认选中上次保存分辨率
- 切换调速到 2x 后重启：启动时自动恢复到 2x

## 6. 边界与异常

- settings.json 不存在：使用系统当前分辨率作为默认
- settings.json 内容非法：忽略并使用默认；不会崩溃
